  1. Add Jobs: Click Add Job to enter job details manually, or use Import Job to extract details from a job posting URL or description.
  2. Track Status: Update the status of each job (saved, applied, interviewing, offered, rejected, etc.) with automatic status history tracking.
  3. Manage Resumes: Create and edit multiple resumes using the integrated JSON Resume standard editor with live preview.
  4. AI-Powered Features: Use advanced AI to tailor resumes, generate cover letters, and analyze job-resume compatibility.
  5. Settings Configuration: Configure API keys for AI services (Claude/OpenAI) and customize themes and preferences.

üìã Interface Overview

  • Jobs: Comprehensive job management with status tracking, contact information, and resume associations. Features tabbed interface for details, contact, resume, and status management.

ü§ñ Advanced AI Features

  • Resume Tailoring: AI analyzes job descriptions and customizes resume content, skills, and experience highlights for maximum compatibility.

üîë API Configuration: Go to Settings to configure your AI provider API keys. Both Claude (recommended) and OpenAI are supported with live testing capabilities.

üí° Tips & Best Practices

  • Data Persistence: All data is stored locally in your browser for privacy. Use export/import features for backup and sharing.
  • Job-Centric Workflow: Start with jobs, then associate tailored resumes and cover letters for a organized approach.
  • Resume Variants: Create multiple resume versions for different job types and use AI to tailor them further.
  • Status Tracking: Keep job statuses current - the system maintains a complete history of status changes.
  • AI History Review: Use the AI History section to review, refine, and re-apply previous AI outputs.
  • Theme Customization: Experiment with different resume themes to find what works best for your industry.
  • Bulk Operations: Use import/export features for managing large numbers of jobs or resumes efficiently.
